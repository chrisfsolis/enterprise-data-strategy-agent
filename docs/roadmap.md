# Roadmap

## Phase 1: Synthetic Domo-style metadata analyzer
Build a polished local demo using mocked metadata and markdown reports.

## Phase 2: Domo read-only connector
Add a secure read-only connector for live metadata collection without write operations.

## Phase 3: Streamlit or lightweight web UI
Create an upload-and-review interface for non-technical data leaders.

## Phase 4: Governance workflow recommendations
Map findings to repeatable governance workflows, ownership reviews, and certification processes.

## Phase 5: AI-assisted natural language strategy assistant
Add grounded Q&A and brief customization over analyzed metadata.

## Phase 6: Additional platform connectors
Support Snowflake, Tableau, Power BI, Looker, dbt, Collibra, Alation, Atlan, and related tools.

## Phase 7: Optional embedded or app-based experiences
Explore embedded experiences for BI administrators and governance teams.
