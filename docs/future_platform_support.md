# Future Platform Support

The same metadata strategy pattern can work across enterprise data ecosystems.

## Snowflake
Ingest tables, views, owners, tags, query usage, freshness, and sensitivity metadata.

## Tableau
Analyze workbooks, dashboards, certified data sources, owners, extracts, and usage.

## Power BI
Review semantic models, reports, workspaces, refresh schedules, endorsements, and ownership.

## Looker
Assess explores, dashboards, LookML metrics, owners, usage, and governed definitions.

## dbt
Use models, exposures, tests, owners, freshness checks, documentation, and semantic layer metadata.

## Collibra
Incorporate governance ownership, stewardship, policy, glossary, and certification context.

## Alation
Use catalog, lineage, trust flags, glossary, popularity, and stewardship metadata.

## Atlan
Connect assets, owners, classifications, lineage, usage, and governance workflows.

## Other tools
Any BI, catalog, governance, or analytics platform with exportable metadata can implement the connector interface.
